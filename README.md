## 🎯 Sobre

Esta é uma landing page profissional demonstrativa do projeto **GreaTIC**, desenvolvida especificamente para apresentação à **Shopee Open Platform**. O site showcassa todas as funcionalidades e benefícios do sistema integrado de monitoramento de ofertas via Discord Bot com IA.

## 🚀 Deploy Automático

O site é automaticamente deployado via **GitHub Pages** sempre que houver push para a branch `main`.

### Configuração do GitHub Pages:
1. Vá em **Settings** > **Pages**
2. Source: **Deploy from a branch**
3. Branch: **main** / **root**
4. Save

